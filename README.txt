======================== MECH WARS =============================
ENGLISH VERSION

ABOUT:

Mech Wars is a Top-down 2D Arena Mech Shooter with Multiplayer. 
Players fight in the arena against other players. 
Players can modify their mechs to suit their playstyle, 
whether it's the type of mech, equipment, etc.

PREREQUISITE SOFTWARE:
- Godot Game Engine 4.4 - https://godotengine.org/download/archive/4.4.1-stable/
- Repository of the Game - https://github.com/TastyMisoSoup/untitled-game/tree/dev

HOW TO OPEN PROJECT FROM REPOSITORY:
- Open Godot Game Engine
- Fetch and switch to dev branch after installing repository
- Import repository as project in Godot
- Open Project in Godot
- After opening the Project, click on the "Run Project" button in upper right corner (alternatively use F5)

SLOVAK VERSION

POPIS:

Mech Wars je Top-down 2D Arena Mech Shooter s možnosťou Multiplayeru. 
Hráči bojujú v aréne proti iným hráčom v štýle Deathmatch. 
Hráči si môžu modifikovať svojich robotov/Mechs aby vyhovovalo ich playstyle-u, 
či už je to typ Mechu, vybavenie, atď.. 

PREDPOKLADY:
- Godot Game Engine 4.4 - https://godotengine.org/download/archive/4.4.1-stable/
- Repozitár hry - https://github.com/TastyMisoSoup/untitled-game/tree/dev

AKO OTVORIŤ PROJEKT:
- Otvorte Godot Game Engine
- Dajte fetch a checkout na dev branch po stiahnutí repozitára
- Importujte repozitár ako projekt do Godot

- Po otvorení projektu, kliknite na "Run Project" tlačidlo v pravom hornom rohu (alternatívne použite F5)
